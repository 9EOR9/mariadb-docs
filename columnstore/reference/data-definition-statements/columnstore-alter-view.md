# ColumnStore Alter View

1. [Syntax "Syntax"](columnstore-alter-view.md#syntax)

Alters the definition of a view. CREATE OR REPLACE VIEW may also be used to alter the definition\
of a view.

## Syntax

```
CREATE
    [OR REPLACE]
    VIEW view_name [(column_list)]
    AS select_statement
```
