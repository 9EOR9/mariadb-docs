# Using MariaDB Replication with MariaDB Galera Cluster

MariaDB Galera Cluster offers synchronous multi-master replication for high availability. Combining it with standard asynchronous replication allows extending data redundancy, typically for disaster recovery to a remote site or for analytical reporting off a dedicated replica, enhancing overall system robustness.
