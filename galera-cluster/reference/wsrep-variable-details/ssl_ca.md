# ssl\_ca

## Overview <a href="#overview_h2" id="overview_h2"></a>

CA file in PEM format (check OpenSSL docs, implies --ssl).

## Details

### Parameters

| Command-line          | --ssl\_ca=arg          |
| --------------------- | ---------------------- |
| Configuration file    | Supported              |
| Dynamic               | No                     |
| Scope                 | Global                 |
| Data Type             | VARCHAR                |
| Product Default Value | "" _(an empty string)_ |
