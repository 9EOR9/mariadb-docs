| Error Code | SQLSTATE | Error                   | Description                                                          |
| ---------- | -------- | ----------------------- | -------------------------------------------------------------------- |
| Error Code | SQLSTATE | Error                   | Description                                                          |
| 4206       |          | ER\_VEC\_DISTANCE\_TYPE | Cannot determine distance type for VEC\_DISTANCE, index is not found |
