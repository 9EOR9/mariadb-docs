# Security

MariaDB Cloud prioritizes security, offering comprehensive protection at every level. This includes encryption in transit, encrypted storage, secure compute, and robust authentication and authorization mechanisms. It adheres to key security standards, such as SOC2, GDPR, and ISO/IEC 27001.. For more information, refer to the remaining sections.
