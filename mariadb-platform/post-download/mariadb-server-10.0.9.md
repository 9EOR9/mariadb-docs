## About this Release

* MariaDB Server 10.0.9 was released on 10 Mar 2014.
* Please note: Newer releases exist within the MariaDB Server 10.0 release series.
* [What's new in this release?](https://app.gitbook.com/s/aEnK0ZXmUbJzqQrTjFyb/mariadb-community-server-release-notes/old-releases/release-notes-mariadb-10-0-series/mariadb-1009-release-notes)

* This release contains [security content](https://app.gitbook.com/s/aEnK0ZXmUbJzqQrTjFyb/mariadb-community-server-release-notes/old-releases/release-notes-mariadb-10-0-series/mariadb-1009-release-notes)

## About this Release Series

* This release is part of the MariaDB Server 10.0 release series.
* The MariaDB Server 10.0 release series will be supported until March 2019.
* [What's new in this release series?](https://app.gitbook.com/s/aEnK0ZXmUbJzqQrTjFyb/mariadb-community-server-release-notes/old-releases/release-notes-mariadb-10-0-series/changes-improvements-in-mariadb-10-0)
