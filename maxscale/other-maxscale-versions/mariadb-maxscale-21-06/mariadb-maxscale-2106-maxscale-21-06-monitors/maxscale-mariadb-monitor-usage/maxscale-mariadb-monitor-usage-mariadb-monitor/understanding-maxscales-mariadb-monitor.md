# Understanding MaxScale's MariaDB Monitor

MaxScale's [MariaDB Monitor](../../../../mariadb-maxscale-23-02/mariadb-maxscale-23-02-monitors/mariadb-maxscale-2302-mariadb-monitor.md) (mariadbmon) monitors [MariaDB replication](https://app.gitbook.com/s/SsmexDFPv2xG2OTyO5yV/ha-and-performance/standard-replication) deployments.

## What Does the MariaDB Monitor Support?

The [MariaDB Monitor (mariadbmon)](../../../../mariadb-maxscale-23-02/mariadb-maxscale-23-02-monitors/mariadb-maxscale-2302-mariadb-monitor.md) supports:

* Monitoring [MariaDB replication](https://app.gitbook.com/s/SsmexDFPv2xG2OTyO5yV/ha-and-performance/standard-replication) deployments
* Query-based load balancing with the Read/Write Split Router (readwritesplit)
* Connection-based load balancing with the Read Connection Router (readconnroute)
