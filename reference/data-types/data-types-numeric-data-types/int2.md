# INT2

`INT2` is a synonym for [SMALLINT](smallint.md).

```
CREATE TABLE t1 (x INT2);

DESC t1;
+-------+-------------+------+-----+---------+-------+
| Field | Type | Null | Key | Default | Extra |
+-------+-------------+------+-----+---------+-------+
| x | smallint(6) | YES | | NULL | |
+-------+-------------+------+-----+---------+-------+
```
