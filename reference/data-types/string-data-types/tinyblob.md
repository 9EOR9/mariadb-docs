# TINYBLOB

#

# Syntax

```
TINYBLOB
```

#

# Description

A [BLOB](blob.md) column with a maximum length of 
255 (28 - 1) bytes. Each
TINYBLOB value is stored using a one-byte length prefix that indicates
the number of bytes in the value.
