This is a [_**Stable**_](../../../community-server/about/release-criteria.md) _**(GA)**_ release of the MariaDB\
Connector/C, formerly known as MariaDB Client Library for C.

**For a description of this library see the**[**MariaDB Connector/C**](https://app.gitbook.com/s/CjGYMsT2MVP4nd3IyW2L/mariadb-connector-c) **page.**

## Notable Bug fixes

* [CONC-152](https://jira.mariadb.org/browse/CONC-152): Build fails with -DWITH\_OPENSSL=On in Visual Studio 2015
* Disable nagle algorithm for tcp/ip connections on Windows by default
* [MDEV-9212](https://jira.mariadb.org/browse/MDEV-9212) ssl-validate-cert incorrect hostname check
* Support for CMake versions >= 3.0
