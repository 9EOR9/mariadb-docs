**Release date:** 19 June 2020

This is a [_**Stable (GA)**_](../../../community-server/about/release-criteria.md) release of the MariaDB\
Connector/C, formerly known as MariaDB Client Library for C.

**For a description of this library see the**[**MariaDB Connector/C**](https://app.gitbook.com/s/CjGYMsT2MVP4nd3IyW2L/mariadb-connector-c) **page.**

## Notable Changes

* [CONC-469](https://jira.mariadb.org/browse/CONC-469): Use servers preferred authentication method unless another plugin was specified.
* Added support for kFreeBSD builds
* Fixed memory leak in GnuTLS if connection couldn't be established.
