**Release date:** 7 Jul 2021

This is a [_**Stable (GA)**_](../../../mariadb-release-criteria.md) release of MariaDB\
Connector/C, formerly known as MariaDB Client Library for C.

**For a description of this library see the**[**MariaDB Connector/C**](https://app.gitbook.com/s/CjGYMsT2MVP4nd3IyW2L/mariadb-connector-c) **page.**

## Notable changes

* First stable (GA) release in the Connector/C 3.2 series
* [CONC-558](https://jira.mariadb.org/browse/CONC-558): Allow all curves when specifying cipher suite (GnuTLS)
* [CONC-503](https://jira.mariadb.org/browse/CONC-503): Add support for OpenSSL 3.0
