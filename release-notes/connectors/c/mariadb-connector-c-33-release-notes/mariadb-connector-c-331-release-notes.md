This is a [_**Stable (GA)**_](../../../mariadb-release-criteria.md) release of MariaDB\
Connector/C, formerly known as MariaDB Client Library for C.

**For a description of this library see the**[**MariaDB Connector/C**](https://app.gitbook.com/s/CjGYMsT2MVP4nd3IyW2L/mariadb-connector-c) **page.**

## Notable changes

* Updated bundled zlib (Version 1.2.12)
* [CONC-580](https://jira.mariadb.org/browse/CONC-580): License header fixes
* [CONC-588](https://jira.mariadb.org/browse/CONC-588): Save proxy settings on reconnect
* Android build fix
* Fix sigbus error with non blocking API (ARM/MacOS)
