# MariaDB Connector/J 2.5.1 Release Notes

The most recent [_**Stable**_](../../../mariadb-release-criteria.md) _**(GA)**_ release of [MariaDB Connector/J](https://github.com/mariadb-corporation/docs-release-notes/blob/test/en/about-mariadb-connector-j/README.md) is:[**MariaDB Connector/J 3.5.3**](../mariadb-connector-j-3-5-release-notes/mariadb-connector-j-3-5-3-release-notes.md)

[Download](https://downloads.mariadb.com/Connectors/java/connector-java-2.5.1)[Release Notes](mariadb-connector-j-251-release-notes.md)[Changelog](../changelogs/mariadb-connector-j-25-changelogs/mariadb-connector-j-251-changelog.md)[Connector/J Overview](https://github.com/mariadb-corporation/docs-release-notes/blob/test/en/about-mariadb-connector-j/README.md)

**Release date:** 16 Oct 2019

MariaDB Connector/J 2.5.1 is a [_**Release Candidate (RC)**_](../../../mariadb-release-criteria.md) release.

**NOTE:** MariaDB Connector/J 2.5.1 is fully compatible with the latest release of\
version 2.4. Further maintenance releases will not be provided for version 2.4.

**For an overview of MariaDB Connector/J see the**[**About MariaDB Connector/J**](https://github.com/mariadb-corporation/docs-release-notes/blob/test/en/about-mariadb-connector-j/README.md) **page**

## Notable Updates

* [CONJ-737](https://jira.mariadb.org/browse/CONJ-737) - Error packet caching\_sha2\_password not handled when not having a password
* [CONJ-738](https://jira.mariadb.org/browse/CONJ-738) - PAM authentication multiple exchanges permitting multiple steps in connection string
* [CONJ-735](https://jira.mariadb.org/browse/CONJ-735) - Multi insert regression correction returning multi generated keys
* [CONJ-736](https://jira.mariadb.org/browse/CONJ-736) - OSGI compliance
