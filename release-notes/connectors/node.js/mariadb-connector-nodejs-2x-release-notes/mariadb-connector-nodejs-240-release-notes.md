This version removes compatibility with Node.js 6, needing 10+ version!

* [CONJS-86](https://jira.mariadb.org/browse/CONJS-86) - Support extension type format
* [CONJS-128](https://jira.mariadb.org/browse/CONJS-128) - Error when using multipleStatements with metaAsArray
* [CONJS-129](https://jira.mariadb.org/browse/CONJS-129) - Support 10.5 pluggable type
* [CONJS-131](https://jira.mariadb.org/browse/CONJS-131) - checkDuplicate option is indicated in error when having dupplicate
* [CONJS-132](https://jira.mariadb.org/browse/CONJS-132) - performance improvement
* [CONJS-136](https://jira.mariadb.org/browse/CONJS-136) - typescript SqlError interface
