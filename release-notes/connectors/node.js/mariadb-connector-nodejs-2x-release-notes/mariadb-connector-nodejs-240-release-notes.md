# MariaDB Connector/Node.js 2.4.0 Release Notes

The most recent [_**Stable (GA)**_](../../../mariadb-release-criteria.md) release of [MariaDB Connector/Node.js](https://github.com/mariadb-corporation/docs-release-notes/blob/test/en/about-mariadb-connector-nodejs/README.md) is:[**MariaDB Connector/Node.js 3.4.2**](../mariadb-connector-nodejs-3x-release-notes/mariadb-connector-node-js-3-4-2-release-notes.md)

**Release date:** 26 May 2020

MariaDB Connector/Node.js 2.4.0 is a [_**Stable**_](../../../mariadb-release-criteria.md) _**(GA)**_ release.

**For an overview of MariaDB Connector/Node.js see the**[**About MariaDB Connector/Node.js**](https://github.com/mariadb-corporation/docs-release-notes/blob/test/en/about-mariadb-connectornodejs/README.md) **page**

This version removes compatibility with Node.js 6, needing 10+ version!

* [CONJS-86](https://jira.mariadb.org/browse/CONJS-86) - Support extension type format
* [CONJS-128](https://jira.mariadb.org/browse/CONJS-128) - Error when using multipleStatements with metaAsArray
* [CONJS-129](https://jira.mariadb.org/browse/CONJS-129) - Support 10.5 pluggable type
* [CONJS-131](https://jira.mariadb.org/browse/CONJS-131) - checkDuplicate option is indicated in error when having dupplicate
* [CONJS-132](https://jira.mariadb.org/browse/CONJS-132) - performance improvement
* [CONJS-136](https://jira.mariadb.org/browse/CONJS-136) - typescript SqlError interface
