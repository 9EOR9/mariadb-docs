* [CONJS-139](https://jira.mariadb.org/browse/CONJS-139) - createConnection(string)\` does not support URL-encoded credentials
* [CONJS-140](https://jira.mariadb.org/browse/CONJS-140) - Support passing null values in array when doing queries. thanks to @koendeschacht
* [CONJS-141](https://jira.mariadb.org/browse/CONJS-141) - set default value of option restoreNodeTimeout to 1000 to avoid using blacklisted pool in cluster
