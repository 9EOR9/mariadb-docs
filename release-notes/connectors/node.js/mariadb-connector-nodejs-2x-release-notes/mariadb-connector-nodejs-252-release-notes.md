## Notable Changes

* \[[CONJS-151](https://jira.mariadb.org/browse/CONJS-151)] bulk batch error (parameter truncation) #137

* \[[CONJS-154](https://jira.mariadb.org/browse/CONJS-154)] Timezone support correction and clarification
* \[[CONJS-155](https://jira.mariadb.org/browse/CONJS-155)] correction to support for node.js 10.13 to 10.19
* \[[CONJS-156](https://jira.mariadb.org/browse/CONJS-156)] Ensure setting capability PLUGIN\_AUTH only if server has it
* documentation improvement
