## Notable changes

* [CONJS-262](https://jira.mariadb.org/browse/CONJS-262) Binary result-set parsing performance improvement, avoiding to chromium slow issue [detail?id=7161](https://bugs.chromium.org/p/v8/issues/detail?id=7161)
* [CONJS-265](https://jira.mariadb.org/browse/CONJS-265) permit configuration of console warning message to be exported
* [CONJS-266](https://jira.mariadb.org/browse/CONJS-266) Option `infileStreamFactory` addition for compatibility
