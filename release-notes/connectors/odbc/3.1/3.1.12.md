# Connector/ODBC 3.1.12 Release Notes

The most recent [_**Stable**_](../../../community-server/about/release-criteria.md) _**(GA)**_ release of MariaDB Connector/ODBC is:[**MariaDB Connector/ODBC 3.2.5**](../3.2/3.2.5.md)

[Download](https://mariadb.com/downloads/#connectors)[Release Notes](3.1.12.md)[Changelog](../changelogs/3.1/3.1.12.md)[About MariaDB Connector/ODBC](https://github.com/mariadb-corporation/docs-release-notes/blob/test/kb/en/about-mariadb-connector-odbc/README.md)

**Release date:** 21 Apr 2021

This is a [Stable (GA)](../../../community-server/about/release-criteria.md) release of MariaDB Connector/ODBC 3.1.

MariaDB Connector/ODBC 3.1.12 is built on top of [MariaDB Connector/C v.3.1.12](../../c/3.1/3.1.12.md).

## Bug Fixes

* [ODBC-305](https://jira.mariadb.org/browse/ODBC-305) - Incorrect insert values for datatype decimal
* [ODBC-308](https://jira.mariadb.org/browse/ODBC-308) - Various issues with SQL\_NUMERIC type conversions
* [ODBC-309](https://jira.mariadb.org/browse/ODBC-309) - Type Conversion Warnings are not returned after execution
* [ODBC-310](https://jira.mariadb.org/browse/ODBC-310) - Connector would connect to the socket if SERVER is specified as localhost on Linux
