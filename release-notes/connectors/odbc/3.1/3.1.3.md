# Connector/ODBC 3.1.3 Release Notes

The most recent [_**Stable**_](../../../community-server/about/release-criteria.md) _**(GA)**_ release of MariaDB Connector/ODBC is:[**MariaDB Connector/ODBC 3.2.5**](../3.2/3.2.5.md)

[Download](https://mariadb.com/downloads/#connectors)[Release Notes](3.1.3.md)[Changelog](../changelogs/3.1/3.1.3.md)[About MariaDB Connector/ODBC](https://github.com/mariadb-corporation/docs-release-notes/blob/test/kb/en/about-mariadb-connector-odbc/README.md)

**Release date:** 8 Aug 2019

This is a [Stable (GA)](../../../community-server/about/release-criteria.md) release of MariaDB Connector/ODBC 3.1.

MariaDB Connector/ODBC 3.1.3 is built on top of [MariaDB Connector/C v.3.1.3](../../c/3.1/3.1.3.md).

## Notable Changes

* [ODBC-226](https://jira.mariadb.org/browse/ODBC-226) - Driver doesn't cause an error when it should
* [ODBC-261](https://jira.mariadb.org/browse/ODBC-261) - Connector deploys direct execution, even for server versions, that do not support it
* [ODBC-262](https://jira.mariadb.org/browse/ODBC-262) - MSI and PKG packages do not contain ed25519 plugin
