# Connector/ODBC 3.1.9 Release Notes

The most recent [_**Stable**_](../../../community-server/about/release-criteria.md) _**(GA)**_ release of MariaDB Connector/ODBC is:[**MariaDB Connector/ODBC 3.2.5**](../3.2/3.2.5.md)

[Download](https://mariadb.com/downloads/#connectors)[Release Notes](3.1.9.md)[Changelog](../changelogs/3.1/3.1.9.md)[About MariaDB Connector/ODBC](https://github.com/mariadb-corporation/docs-release-notes/blob/test/en/about-mariadb-connector-odbc/README.md)

**Release date:** 30 Jun 2020

This is a [Stable (GA)](../../../community-server/about/release-criteria.md) release of MariaDB Connector/ODBC 3.1.

MariaDB Connector/ODBC 3.1.9 is built on top of [MariaDB Connector/C v.3.1.9](../../c/3.1/3.1.9.md).

Please note, there was no Connector/ODBC 3.1.8 release. It has been skipped to stay in sync with Connector/C.

## Bug Fixes

* [ODBC-278](https://jira.mariadb.org/browse/ODBC-278) - Can not install Connector/ODBC's Debian binary tarball on Debian Stretch/Buster and Ubuntu Eoan/Focal
* [ODBC-279](https://jira.mariadb.org/browse/ODBC-279) - Binding parameter as SQL\_C\_TIME does not work correctly in some cases
* [ODBC-280](https://jira.mariadb.org/browse/ODBC-280) - Process crashes after some time
* [ODBC-283](https://jira.mariadb.org/browse/ODBC-283) - Connector does not support passphrase for a passphrase-protected private key
* [ODBC-284](https://jira.mariadb.org/browse/ODBC-284) - MariaDB Connector/ODBC does not support closing curly brace (}) in connection keyword (aka paramter)
* [ODBC-285](https://jira.mariadb.org/browse/ODBC-285) - Connector/ODBC binary tarballs tarbomb the user
* [ODBC-287](https://jira.mariadb.org/browse/ODBC-287) - Cannot find mysql.h
* [ODBC-214](https://jira.mariadb.org/browse/ODBC-214) - Mediumint(INT24) is always empty in Visual FoxPro
