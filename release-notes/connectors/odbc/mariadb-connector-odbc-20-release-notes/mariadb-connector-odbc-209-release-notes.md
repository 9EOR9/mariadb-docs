# Connector/ODBC 2.0.9 Release Notes

The most recent [_**Stable**_](../../../community-server/about/release-criteria.md) _**(GA)**_ release of MariaDB Connector/ODBC is:[**MariaDB Connector/ODBC 3.2.5**](../mariadb-connector-odbc-3-2-release-notes/mariadb-connector-odbc-3-2-5-release-notes.md)

**Release date:** 16 Nov 2015

This is a [_**Beta**_](../../../community-server/about/release-criteria.md) release of MariaDB\
Connector/ODBC.

MariaDB Connector/ODBC 2.0.9 is built on top of [MariaDB Connector/C 2.2](../../c/mariadb-connector-c-22-release-notes/mariadb-connector-c-220-release-notes.md) and uses the\
binary prepared statement protocol. It is basically MariaDB Connector/ODBC 1.0\
up to version [1.0.5](../mariadb-connector-odbc-10-release-notes/mariadb-connector-odbc-105-release-notes.md), ported to work on Linux.
