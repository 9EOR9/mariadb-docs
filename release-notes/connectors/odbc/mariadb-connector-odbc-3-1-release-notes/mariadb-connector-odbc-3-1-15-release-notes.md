The most recent [_**Stable**_](../../../community-server/about/release-criteria.md) _**(GA)**_ release of MariaDB Connector/ODBC is:[**MariaDB Connector/ODBC 3.2.5**](../mariadb-connector-odbc-3-2-release-notes/mariadb-connector-odbc-3-2-5-release-notes.md)

**Release date:** 18 Nov 2021

This is a [Stable (GA)](../../../community-server/about/release-criteria.md) release of MariaDB Connector/ODBC 3.1.

MariaDB Connector/ODBC 3.1.15 is built on top of [MariaDB Connector/C v.3.2.5](../../c/3.2/3.2.5.md).

## Bug Fixes

* [ODBC-343](https://jira.mariadb.org/browse/ODBC-343) - MariaDB ODBC 3.1.14 setup dialog crashes at the attempt to add new DSN. That was a regression introduced in the [version 3.1.14](mariadb-connector-odbc-3-1-14-release-notes.md)
* [ODBC-345](https://jira.mariadb.org/browse/ODBC-345) - Date values beyond TIMESTAMP range would not be saved in the DATETIME field.
