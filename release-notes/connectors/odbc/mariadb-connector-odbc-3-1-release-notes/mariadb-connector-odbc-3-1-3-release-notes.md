# MariaDB Connector/ODBC 3.1.3 Release Notes

The most recent [_**Stable**_](../../../mariadb-release-criteria.md) _**(GA)**_ release of MariaDB Connector/ODBC is:[**MariaDB Connector/ODBC 3.2.5**](../mariadb-connector-odbc-3-2-release-notes/mariadb-connector-odbc-3-2-5-release-notes.md)

**Release date:** 8 Aug 2019

This is a [Stable (GA)](../../../mariadb-release-criteria.md) release of MariaDB Connector/ODBC 3.1.

MariaDB Connector/ODBC 3.1.3 is built on top of[MariaDB Connector/C v.3.1.3](../../c/mariadb-connector-c-31-release-notes/mariadb-connector-c-313-release-notes.md).

## Notable Changes

* [ODBC-226](https://jira.mariadb.org/browse/ODBC-226) - Driver doesn't cause an error when it should
* [ODBC-261](https://jira.mariadb.org/browse/ODBC-261) - Connector deploys direct execution, even for server versions, that do not support it
* [ODBC-262](https://jira.mariadb.org/browse/ODBC-262) - MSI and PKG packages do not contain ed25519 plugin
