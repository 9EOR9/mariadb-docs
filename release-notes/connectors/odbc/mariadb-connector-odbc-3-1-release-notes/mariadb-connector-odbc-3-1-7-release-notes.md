This is a [Stable (GA)](../../../community-server/about/release-criteria.md) release of MariaDB Connector/ODBC 3.1.

MariaDB Connector/ODBC 3.1.7 is built on top of [MariaDB Connector/C v.3.1.7](../../c/3.1/3.1.7.md).

## Updates

* [ODBC-270](https://jira.mariadb.org/browse/ODBC-270) - The ODBC driver does not compile from sources on Centos 8
* [ODBC-271](https://jira.mariadb.org/browse/ODBC-271) - MSI should not install on Windows 7
* [ODBC-276](https://jira.mariadb.org/browse/ODBC-276) - Update of Binary data containing 0x00 does not work
* [ODBC-277](https://jira.mariadb.org/browse/ODBC-277) - Memory Corruption from Incorrect Type for SQL\_IDENTIFIER\_CASE
* [ODBC-274](https://jira.mariadb.org/browse/ODBC-274) - Support [INSERT/REPLACE ... RETURNING](https://app.gitbook.com/s/SsmexDFPv2xG2OTyO5yV/reference/sql-statements/data-manipulation/inserting-loading-data/insertreturning) (requires [MariaDB 10.5](../../../community-server/old-releases/mariadb-10-5-series/what-is-mariadb-105.md))
