**Release date:** 30 Sep 2013

[MariaDB Galera 5.5.33](https://mariadb.com/kb/en/mariadb-galera-cluster-5533-release-notes/)a is a [_**Stable**_](../../../../mariadb-release-criteria.md) (GA) release.\
It is a merge of [MariaDB 5.5.33a](broken-reference) and[Galera Cluster](https://codership.com/content/using-galera-cluster) with\
additional bug fixes.

Various articles about MariaDB Galera Cluster, including[known limitations](https://app.gitbook.com/s/3VYeeVGUV4AMqrA3zwy7/galera-management/mariadb-galera-cluster-known-limitations) and[how to get started](https://app.gitbook.com/s/3VYeeVGUV4AMqrA3zwy7/galera-management/getting-started-with-mariadb-galera-cluster) are\
available in the [**Galera**](https://github.com/mariadb-corporation/docs-release-notes/blob/test/kb/en/galera/README.md) section of the Knowledgebase.

For a list of changes made in [MariaDB Galera 5.5.33](https://github.com/mariadb-corporation/docs-release-notes/blob/test/kb/en/mariadb-galera-cluster-5533-release-notes/README.md)a, with links to detailed\
information on each push, see the[MariaDB Galera 5.5.33a Changelog](../mariadb-galera-55-changelogs/mariadb-galera-5533a-changelog.md).

## Includes [MariaDB 5.5.33a](broken-reference) and Galera Cluster

[MariaDB Galera 5.5.33](https://github.com/mariadb-corporation/docs-release-notes/blob/test/kb/en/mariadb-galera-cluster-5533-release-notes/README.md)a includes [MariaDB 5.5.33a](broken-reference) with Codership additions and[Galera Cluster 23.2.7](https://codership.com/content/using-galera-cluster). See the[MariaDB 5.5.33a Release Notes](broken-reference) and[Changelog](../../../../changelogs/changelogs-mariadb-55-series/mariadb-5533a-changelog.md) for more information on the changes in\
MariaDB.

Thanks, and enjoy MariaDB Galera Cluster!
