# mysqld\_safe

From [MariaDB 10.5](https://app.gitbook.com/s/aEnK0ZXmUbJzqQrTjFyb/mariadb-community-server-release-notes/mariadb-10-5-series/what-is-mariadb-105), the client is called `mariadbd-safe`. It can still be accessed under its original `mysqld_safe` name via a symlink in Linux, or an alternate binary in Windows.

See [mariadbd-safe](../../server-management/starting-and-stopping-mariadb/mariadbd-safe.md) for details.
