# mysql\_plugin

`mysql_plugin` is a tool for enabling or disabling [plugins](../../reference/plugins/).

From [MariaDB 10.5](https://app.gitbook.com/s/aEnK0ZXmUbJzqQrTjFyb/mariadb-community-server-release-notes/mariadb-10-5-series/what-is-mariadb-105), the client is called `mariadb-plugin`. It can still be accessed under its original `mysql_plugin` name via a symlink in Linux, or an alternate binary in Windows.

See [mariadb-plugin](../administrative-tools/mariadb-plugin.md) for details.
