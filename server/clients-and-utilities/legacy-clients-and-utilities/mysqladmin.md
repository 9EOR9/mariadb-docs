# mysqladmin

mysqladmin is an administration program for the mysqld daemon.

From [MariaDB 10.5](https://app.gitbook.com/s/aEnK0ZXmUbJzqQrTjFyb/mariadb-community-server-release-notes/mariadb-10-5-series/what-is-mariadb-105), the client is called `mariadb-admin`. It can still be accessed under its original `mysqladmin` name via a symlink in Linux, or an alternate binary in Windows.

See [mariadb-admin](../administrative-tools/mariadb-admin.md) for details.
