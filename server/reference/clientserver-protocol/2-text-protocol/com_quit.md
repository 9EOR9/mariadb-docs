# COM\_QUIT

Using the `COM_QUIT` command, the client tells the server that the connection should be terminated.

## Fields

* [int<1>](../protocol-data-types.md#fixed-length-integers) `0x01` : `COM_QUIT` header.

## Response

Server terminates connection.
