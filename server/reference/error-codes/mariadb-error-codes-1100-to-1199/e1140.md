| Error Code | SQLSTATE | Error                                 | Description                                                                                                       |
| ---------- | -------- | ------------------------------------- | ----------------------------------------------------------------------------------------------------------------- |
| 1140       | 42000    | ER\_MIX\_OF\_GROUP\_FUNC\_AND\_FIELDS | Mixing of GROUP columns (MIN(),MAX(),COUNT(),...) with no GROUP columns is illegal if there is no GROUP BY clause |
