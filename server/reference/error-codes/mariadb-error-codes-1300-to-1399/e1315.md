# Error 1315: The update log is deprecated and replaced by the binary log

| Error Code | SQLSTATE | Error                                | Description                                                                                                                                                                                                                                     |
| ---------- | -------- | ------------------------------------ | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 1315       | 42000    | ER\_UPDATE\_LOG\_DEPRECATED\_IGNORED | The update log is deprecated and replaced by the binary log; SET SQL\_LOG\_UPDATE has been ignored. This option will be removed in [MariaDB 5.6](https://github.com/mariadb-corporation/docs-server/blob/test/en/what-is-mariadb-56/README.md). |
