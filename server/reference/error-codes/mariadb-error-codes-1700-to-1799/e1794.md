# Error 1794: Slave is not configured or failed to initialize properly. You must at least set --server

| ---------- | -------- | ------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 1794       | HY000    | ER\_SLAVE\_CONFIGURATION | Slave is not configured or failed to initialize properly. You must at least set --server-id to enable either a master or a slave. Additional error messages can be found in the MariaDB error log |
