# What is MariaDB?

MariaDB is a [backward compatible](https://app.gitbook.com/s/aEnK0ZXmUbJzqQrTjFyb/mariadb-community-server/mariadb-releases/compatibility-differences/mariadb-vs-mysql-compatibility), drop-in replacement of the MySQLÂ® Database Server. It includes all major open source storage engines.

You can see a [list of all current and old releases](/en/new-and-old-releases/) as well as a [feature comparison to MySQL](https://app.gitbook.com/s/aEnK0ZXmUbJzqQrTjFyb/mariadb-community-server/mariadb-releases/compatibility-differences/mariadb-vs-mysql-features).

The source code for MariaDB is [publically available](../../../clients-and-utilities/server-client-software/download/getting-the-mariadb-source-code.md) on 
[GitHub](https://github.com/MariaDB/server). Binaries and packages are [also available](http://mariadb.org/download).
