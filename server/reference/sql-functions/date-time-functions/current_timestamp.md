# CURRENT\_TIMESTAMP

## Syntax

```sql
CURRENT_TIMESTAMP
CURRENT_TIMESTAMP([precision])
```

## Description

`CURRENT_TIMESTAMP` and `CURRENT_TIMESTAMP()` are synonyms for [NOW()](now.md).
