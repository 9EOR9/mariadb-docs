
# DAYOFYEAR

## Syntax


```
DAYOFYEAR(date)
```

## Description


Returns the day of the year for date, in the range 1 to 366.


## Examples


```
SELECT DAYOFYEAR('2018-02-16');
+-------------------------+
| DAYOFYEAR('2018-02-16') |
+-------------------------+
|                      47 |
+-------------------------+
```
