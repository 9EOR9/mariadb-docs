# LOCALTIMESTAMP

## Syntax

```sql
LOCALTIMESTAMP
LOCALTIMESTAMP([precision])
```

## Description

`LOCALTIMESTAMP` and `LOCALTIMESTAMP()` are synonyms for [NOW()](now.md).
