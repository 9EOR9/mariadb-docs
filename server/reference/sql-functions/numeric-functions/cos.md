# COS

## Syntax

```sql
COS(X)
```

## Description

Returns the cosine of X, where X is given in radians.

## Examples

```sql
SELECT COS(PI());
+-----------+
| COS(PI()) |
+-----------+
|        -1 |
+-----------+
```
