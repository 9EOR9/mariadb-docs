# COLUMN\_LIST

## Syntax

```sql
COLUMN_LIST(dyncol_blob);
```

## Description

Returns a comma-separated list of column names. The names are quoted with backticks.

See [dynamic columns](../../../sql-structure/nosql/dynamic-columns.md) for more information.
