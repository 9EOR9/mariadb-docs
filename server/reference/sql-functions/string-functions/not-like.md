# NOT LIKE

## Syntax

```
expr NOT LIKE pat [ESCAPE 'escape_char']
```

## Description

This is the same as [NOT (expr LIKE pat \[ESCAPE 'escape\_char'\])](../../sql-structure/operators/logical-operators/not.md).
