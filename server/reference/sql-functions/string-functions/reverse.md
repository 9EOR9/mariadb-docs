
# REVERSE

## Syntax


```
REVERSE(str)
```

## Description


Returns the string `str` with the order of the characters reversed.


## Examples


```
SELECT REVERSE('desserts');
+---------------------+
| REVERSE('desserts') |
+---------------------+
| stressed            |
+---------------------+
```
