# RIGHT

## Syntax

```
RIGHT(str,len)
```

## Description

Returns the rightmost _`len`_ characters from the string _`str`_, or NULL if\
any argument is NULL.

## Examples

```
SELECT RIGHT('MariaDB', 2);
+---------------------+
| RIGHT('MariaDB', 2) |
+---------------------+
| DB                  |
+---------------------+
```
