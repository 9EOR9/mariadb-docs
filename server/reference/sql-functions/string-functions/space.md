
# SPACE

## Syntax


```
SPACE(N)
```

## Description


Returns a string consisting of *`N`* space characters. If `N` is NULL, returns NULL.


## Examples


```
SELECT QUOTE(SPACE(6));
+-----------------+
| QUOTE(SPACE(6)) |
+-----------------+
| '      '        |
+-----------------+
```
