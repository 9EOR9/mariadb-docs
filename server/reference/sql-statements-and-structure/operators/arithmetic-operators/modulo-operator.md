
# Modulo Operator (%)

## Syntax


```
N % M
```

## Description


Modulo operator. Returns the remainder of `<code>N</code>` divided by `<code>M</code>`. See also [MOD](../../sql-statements/built-in-functions/numeric-functions/mod.md).


## Examples


```
SELECT 1042 % 50;
+-----------+
| 1042 % 50 |
+-----------+
|        42 |
+-----------+
```
