
# Information Schema ROCKSDB_CF_OPTIONS Table

The [Information Schema](../../../../../../../mariadb-internals/information-schema-plugins-show-and-flush-statements.md) `<code>ROCKSDB_CF_OPTIONS</code>` table is included as part of the [MyRocks](../../../../../../../storage-engines/myrocks/myrocks-in-mariadb-102-vs-mariadb-103.md) storage engine, and contains infomation about MyRocks [column families](../../../../../../../storage-engines/myrocks/myrocks-column-families.md).


The `<code>PROCESS</code>` [privilege](../../../../../account-management-sql-commands/grant.md) is required to view the table.
