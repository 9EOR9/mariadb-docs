
# Sys Schema


##### MariaDB starting with [10.6.0](https://app.gitbook.com/s/aEnK0ZXmUbJzqQrTjFyb/mariadb-community-server-release-notes/release-notes-mariadb-10-6-series/mariadb-1060-release-notes)
The sys_schema is a collection of views, functions and procedures to help administrators get insight into  database usage.
The [Performance Schema](../performance-schema/README.md) is required to be enabled in order to make use of the Sys Schema.
