
# The mysql Database Tables

MariaDB comes pre-installed with a system database called `mysql` containing many important tables storing, in particular, [grant and privilege](../../../account-management-sql-commands/grant.md) information. Before [MariaDB 10.4](https://app.gitbook.com/s/aEnK0ZXmUbJzqQrTjFyb/mariadb-community-server/old-releases/release-notes-mariadb-10-4-series/what-is-mariadb-104), system tables used the [MyISAM](../../../../../storage-engines/myisam-storage-engine/README.md) storage engine. From [MariaDB 10.4](https://app.gitbook.com/s/aEnK0ZXmUbJzqQrTjFyb/mariadb-community-server/old-releases/release-notes-mariadb-10-4-series/what-is-mariadb-104), they use [Aria](../../../../../storage-engines/aria/README.md).
