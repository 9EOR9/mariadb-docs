
# CURRENT_TIMESTAMP

## Syntax


```
CURRENT_TIMESTAMP
CURRENT_TIMESTAMP([precision])
```

## Description


`<code>CURRENT_TIMESTAMP</code>` and `<code>CURRENT_TIMESTAMP()</code>` are synonyms for `<code>[NOW()](now.md)</code>`.
