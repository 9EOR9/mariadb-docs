
# <<

## Syntax


```
value1 << value2
```

## Description


Converts a longlong ([BIGINT](../../../../../data-types/data-types-numeric-data-types/bigint.md)) number (*value1*) to binary and shifts *value2* units to the left.


## Examples


```
SELECT 1 << 2;
+--------+
| 1 << 2 |
+--------+
|      4 |
+--------+
```
