
# WSREP_LAST_WRITTEN_GTID

## Syntax


```
WSREP_LAST_WRITTEN_GTID()
```

## Description


Returns the [Global Transaction ID](../../../../../../server-usage/replication-cluster-multi-master/standard-replication/gtid.md) of the most recent write transaction performed by the client.
