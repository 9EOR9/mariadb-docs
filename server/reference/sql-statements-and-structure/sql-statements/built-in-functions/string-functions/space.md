
# SPACE

## Syntax


```
SPACE(N)
```

## Description


Returns a string consisting of *`<code>N</code>`* space characters. If `<code>N</code>` is NULL, returns NULL.


## Examples


```
SELECT QUOTE(SPACE(6));
+-----------------+
| QUOTE(SPACE(6)) |
+-----------------+
| '      '        |
+-----------------+
```
