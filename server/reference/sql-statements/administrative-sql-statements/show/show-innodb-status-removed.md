# SHOW INNODB STATUS (removed)

## Syntax

```sql
SHOW INNODB STATUS
```

## Description

This is a deprecated synonym for [SHOW ENGINE INNODB STATUS](show-engine.md). It was removed in MariaDB and MySQL 5.5.
