# Sys Schema

**MariaDB starting with** [**10.6.0**](https://app.gitbook.com/s/aEnK0ZXmUbJzqQrTjFyb/community-server/mariadb-10-6-series/mariadb-1060-release-notes)

The sys\_schema is a collection of views, functions and procedures to help administrators get insight into database usage.\
The [Performance Schema](../performance-schema/) is required to be enabled in order to make use of the Sys Schema.
