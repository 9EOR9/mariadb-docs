# ST\_POINTONSURFACE

## Syntax

```sql
ST_PointOnSurface(g)
PointOnSurface(g)
```

## Description

Given a geometry, returns a [POINT](point.md) guaranteed to intersect a surface. However, see [MDEV-7514](https://jira.mariadb.org/browse/MDEV-7514).

ST\_PointOnSurface() and PointOnSurface() are synonyms.
