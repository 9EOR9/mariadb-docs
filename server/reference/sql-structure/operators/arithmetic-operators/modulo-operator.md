# Modulo Operator (%)

## Syntax

```
N % M
```

## Description

Modulo operator. Returns the remainder of `N` divided by `M`. See also [MOD](../../../sql-functions/numeric-functions/mod.md).

## Examples

```
SELECT 1042 % 50;
+-----------+
| 1042 % 50 |
+-----------+
|        42 |
+-----------+
```
