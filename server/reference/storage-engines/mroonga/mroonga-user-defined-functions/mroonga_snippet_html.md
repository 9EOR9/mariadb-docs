
# mroonga_snippet_html

## Description


`mroonga_snippet_html` is a [user-defined function](../../../../server-usage/programming-customizing-mariadb/user-defined-functions/user-defined-functions-security.md) (UDF) included with the [Mroonga storage engine](mroonga_snippet_html.md). It provides a keyword with surrounding text, or the keyword in context. It is still considered experimental. See [Creating Mroonga User-Defined Functions](creating-mroonga-user-defined-functions.md) for details on creating this UDF if required.
