# S3 Storage Engine

#

#### MariaDB starting with [10.5](https://app.gitbook.com/s/aEnK0ZXmUbJzqQrTjFyb/mariadb-community-server/what-is-mariadb-105)

The [S3 storage engine](s3-storage-engine-status-variables.md) has been available since [MariaDB 10.5.4](https://app.gitbook.com/s/aEnK0ZXmUbJzqQrTjFyb/mariadb-community-server/release-notes-mariadb-105-series/mariadb-1054-release-notes).

S3 is a read-only storage engine that stores its data in Amazon S3.
