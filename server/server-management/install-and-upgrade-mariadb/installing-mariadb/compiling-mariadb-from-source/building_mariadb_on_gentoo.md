# Building MariaDB on Gentoo

MariaDB is in Gentoo, so the steps to build it are:

1.  Synchronize your tree with\


    ```shell
    emerge --sync
    ```
2.  Build MariaDB using\


    ```bash
    emerge mariadb
    ```
