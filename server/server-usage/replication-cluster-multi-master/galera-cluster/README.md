
# MariaDB Galera Cluster

MariaDB Galera Cluster is a [virtually synchronous](about-galera-replication.md) multi-master cluster that runs on Linux only. It has been a standard part of the server since [MariaDB 10.1](https://app.gitbook.com/s/aEnK0ZXmUbJzqQrTjFyb/mariadb-community-server/old-releases/release-notes-mariadb-10-1-series/changes-improvements-in-mariadb-10-1).
