# mroonga\_snippet\_html

## Description

`mroonga_snippet_html` is a [user-defined function](../../../../server-usage/user-defined-functions/) (UDF) included with the [Mroonga storage engine](../). It provides a keyword with surrounding text, or the keyword in context. It is still considered experimental. See [Creating Mroonga User-Defined Functions](creating-mroonga-user-defined-functions.md) for details on creating this UDF if required.
