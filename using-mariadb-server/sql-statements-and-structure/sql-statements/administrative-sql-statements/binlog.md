The `BINLOG` statement in MariaDB is used internally by the `mariadb-binlog` program to translate events from binary log files into a printable form. ### Description <a href="#description" id="description"></a>

`BINLOG` is an internal-use statement. It is generated by the [mariadb-binlog](https://mariadb.com/kb/en/mariadb-binlog/) program as the printable representation of certain events in [binary log](https://mariadb.com/kb/en/binary-log/) files. The `'str'` value is a base 64-encoded string that the server decodes to determine the data change indicated by the corresponding event.&#x20;
